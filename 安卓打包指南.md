## 📥 安装步骤

### 1. 安装Python依赖

打开命令提示符（CMD），执行：

```bash
pip install buildozer kivy
```

### 2. 安装Java JDK

下载并安装 JDK 11：
https://www.oracle.com/java/technologies/javase-jdk11-downloads.html

安装后设置环境变量：
```
JAVA_HOME = C:\Program Files\Java\jdk-11
```

### 3. 安装Android SDK和NDK

#### 方法一：使用Buildozer自动安装（推荐）

首次运行打包命令时，Buildozer会自动下载所需的Android SDK和NDK。

#### 方法二：手动安装

1. 下载 Android Command Line Tools：
https://developer.android.com/studio#command-line-tools-only

2. 安装SDK：
```bash
sdkmanager "platforms;android-28" "build-tools;28.0.3"
```

3. 安装NDK：
```bash
sdkmanager "ndk;21.4.7075529"
```

4. 设置环境变量：
```
ANDROID_HOME = C:\Users\你的用户名\AppData\Local\Android\Sdk
ANDROID_NDK_HOME = C:\Users\你的用户名\AppData\Local\Android\Sdk\ndk\21.4.7075529
```

## 📦 打包方法

### 方法一：一键打包（推荐）

双击运行项目中的 **`一键打包安卓.bat`** 文件。

### 方法二：命令行打包

```bash
# 进入项目目录
cd d:\python学习\Baimuchen_shuxueqimeng

# 初始化Buildozer
buildozer init

# 打包Debug版本
buildozer android debug

# 打包Release版本（需要签名）
buildozer android release
```

### 方法三：使用打包脚本

编辑 `一键打包安卓.bat` 内容如下：

```batch
@echo off
chcp 65001 >nul
echo ============================================
echo     宝贝数学启蒙 - 安卓打包脚本
echo ============================================
echo.

:: 检查并安装依赖
python -c "import buildozer" >nul 2>&1
if errorlevel 1 (
    echo [提示] 正在安装Buildozer...
    pip install buildozer kivy
)

:: 清理旧构建
echo [步骤1] 清理旧构建文件...
if exist .buildozer rd /s /q .buildozer
if exist bin rd /s /q bin

:: 开始打包
echo [步骤2] 开始打包APK（可能需要10-30分钟）...
buildozer android debug

:: 检查结果
if exist bin\*.apk (
    echo.
    echo ============================================
    echo     打包成功！
    echo ============================================
    dir bin\*.apk
) else (
    echo [错误] 打包失败
)

pause
```

## ❓ 常见问题

### Q1: 打包时间太长怎么办？
**A**: 首次打包需要下载Android SDK（约2GB），请耐心等待。后续打包会快很多。

### Q2: 提示"Python not found"？
**A**: 确保Python已安装并添加到系统PATH环境变量。

### Q3: 提示"Android SDK not found"？
**A**: 
1. 安装Android Studio并下载SDK
2. 或手动下载Command Line Tools
3. 设置 `ANDROID_HOME` 环境变量

### Q4: 打包失败，提示权限错误？
**A**: 以管理员身份运行命令提示符或打包脚本。

### Q5: APK安装后闪退？
**A**: 
1. 检查日志：`buildozer android debug 2>&1 | tee debug.log`
2. 可能是Python代码错误或缺少依赖
3. 确保 `main.py` 中没有PC端专用代码（如 pyttsx3）

## 📁 项目文件说明

### 核心文件
```
Baimuchen_shuxueqimeng/
├── main.py                 # 主程序（PC端+安卓通用）
├── config.py              # 配置文件
├── game_logic.py          # 游戏逻辑
├── question_types.py      # 题目类型
├── buildozer.spec         # Buildozer配置 ← 打包必需
├── 一键打包安卓.bat        # 一键打包脚本 ← 双击运行
└── bin/                   # APK输出目录（打包后生成）
```

### buildozer.spec 关键配置
```ini
[app]
title = 宝贝数学启蒙                    # 应用名称
package.name = baimuchen_math          # 包名（小写+下划线）
package.domain = com.baimuchen         # 域名
version = 1.0                          # 版本号
requirements = python3,kivy             # 依赖库
orientation = portrait                 # 竖屏模式

[buildozer]
android.minapi = 24                    # 最低Android版本
android.api = 28                       # 目标Android版本
```

2. 打开手机设置 → 安全 → 允许安装未知来源应用
3. 点击APK文件安装

## 🔒 生成正式签名APK（发布用）

```bash
# 1. 生成密钥库
keytool -genkey -v -keystore my-release-key.jks -keyalg RSA -keysize 2048 -validity 10000 -alias my-key-alias

# 2. 打包Release版本
buildozer android release

# 3. 使用密钥签名
jarsigner -verbose -sigalg SHA1withRSA -digestalg SHA1 -keystore my-release-key.jks bin/*-release-unsigned.apk my-key-alias

# 4. 对齐优化
zipalign -v 4 bin/*-release-unsigned.apk bin/*-release-signed.apk
```
